# TUREG Sign Templater (Word Add-in)

เครื่องมือช่วยจัดการและพิมพ์ข้อมูลลงบนซองจดหมายหรือเอกสารมาตรฐานของกองทะเบียนฯ (Thammasat Registrar) โดยอัตโนมัติ ไม่ต้องเสียเวลาจัดฟอร์แมตหรือพิมพ์ที่อยู่เดิมๆ ซ้ำๆ

## 🌟 ทำอะไรได้บ้าง?
- **ดึงแม่แบบอัตโนมัติ**: แค่เปิด Word เปล่าๆ แล้วกดปุ่ม ระบบจะดึงรูปแบบซอง (C5) และ Placeholder มาวางให้ทันที
- **ที่อยู่ด่วน**: มีรายชื่อสถาบันยอดนิยม เช่น LSAC, Harvard, มหาวิทยาลัยในต่างประเทศ เลือกแล้วที่อยู่จะกรอกให้เอง
- **ระบบแก้ไขยืดหยุ่น**: เมื่อเติมข้อมูลไปแล้ว สามารถเปลี่ยนที่อยู่หรือติ๊กรายการเอกสารเพิ่ม/ลด แล้วกดอัปเดตในหน้าเดิมได้ทันที (ฟอร์แมตไม่พัง)
- **รองรับรหัสพิเศษ**: มีช่องสำหรับใส่ BOLE ID หรือ LSAC Number โดยเฉพาะ ตามข้อกำหนดของแต่ละสถาบัน

---

## 🛠 วิธีการติดตั้ง (สำหรับผู้ใช้งานทั่วไป)

เนื่องจากเป็นเครื่องมือเสริมสำหรับ Microsoft Word (VSTO Add-in) มีขั้นตอนการติดตั้งง่ายๆ ดังนี้:

1.  **ดาวน์โหลดไฟล์**: ดาวน์โหลดโฟลเดอร์โปรเจกต์นี้ลงมาในเครื่องคอมพิวเตอร์
2.  **เปิดใช้งานสิทธิ์ (Trust Folder)**:
    *   เนื่องจาก Windows มีระบบความปลอดภัย ให้คลิกขวาที่โฟลเดอร์ที่โหลดมา -> เลือก **Properties**
    *   ถ้ามีคำว่า **Unblock** ให้ติ๊กถูกแล้วกด OK
3.  **รันไฟล์ติดตั้ง**:
    *   เข้าไปในโฟลเดอร์ มองหาไฟล์ที่ชื่อว่า `register.ps1`
    *   คลิกขวาที่ไฟล์นั้น แล้วเลือก **Run with PowerShell**
    *   รอจนระบบทำงานเสร็จ (เพื่อลงทะเบียนเครื่องมือเข้ากับโปรแกรม Word ในเครื่อง)
4.  **เปิด Word**: เปิดโปรแกรม Microsoft Word ขึ้นมา คุณจะเห็นแถบเมนูใหม่หรือหน้าต่างทางด้านขวาชื่อ **"TUREG Tools"**

---

## 📖 วิธีใช้งานเบื้องต้น

1.  **เปิดแถบเครื่องมือ**: ถ้ายังไม่เห็นหน้าต่างด้านขวา ให้มองหาเมนู **TUREG Add-in** ด้านบนแล้วกดปุ่มเพื่อเปิด
2.  **เลือกสิ่งที่ต้องการ**:
    *   เลือก **ระดับการศึกษา** (Bachelor / Master / etc.)
    *   เลือก **ที่อยู่ปลายทาง** จากรายการด่วน (หรือพิมพ์เองในช่องด้านล่าง)
    *   กรอก **Admission No.** ของนักศึกษา
3.  **ติ๊กรายการเอกสาร**: เลือกเอกสารที่บรรจุลงซอง (ระบบจะใส่เลข "1" ต่อท้ายให้เองอัตโนมัติ)
4.  **กดปุ่ม Fill / Update**: 
    *   ถ้าเป็นเอกสารว่าง ระบบจะสร้างเทมเพลตขึ้นมาให้
    *   ถ้ามีข้อมูลอยู่แล้ว ระบบจะอัปเดตข้อมูลในจุดเดิมให้ทันที

---

## ⚠️ ข้อควรระวัง
- รองรับ Microsoft Word บน Windows เท่านั้น
- หากมีการย้ายโฟลเดอร์โปรเจกต์ไปที่อื่น ต้องรันไฟล์ `register.ps1` ใหม่อีกครั้งเพื่อให้ Word หาไฟล์เจอครับ
